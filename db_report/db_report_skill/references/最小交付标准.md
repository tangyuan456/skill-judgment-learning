> 4 类报告的差异在 A6（图表数量基线）和 D6（结论数量基线）两项。

## E. 排版质量（9 项）

- E3.1 完整 head + 内联 CSS

## F. 占位符与一致性（3 项）

- F1 所有 `{X}` 占位符已替换（md / html grep 不到 `\{[A-Za-z_]+\}`）
- F2 调优建议中的具体数值非默认占位符
- F3 三格式核心数字一致（抽样关键 QPS 值在 md / html 都存在）

## 失败处理流程

任意 ❌ → 立即停止 → 输出失败清单 → 按下表回到对应阶段修复 → 修复后重新核查：

| 失败类型 | 回到阶段 |
|---------|---------|
| A 类（文件） | 阶段 2 / 4 / 5 对应产出脚本 |
| B 类（数据）| 阶段 2 重新接入 |
| C 类（图表）| 阶段 4 重新生成 |
| D 类（分析）| 阶段 3 重新归纳 |
| E 类（排版）| 阶段 5 修复渲染器 |
| F 类（占位）| `templates/{type}_report.py` 替换逻辑 |
