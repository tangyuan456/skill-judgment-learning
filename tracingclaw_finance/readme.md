# TracingClaw Finance 学生版说明

这个目录提供一个学生可用的金融验真 Skill。学生环境不能访问内部链路接口，因此本版本不做内部搜索链路还原，只做金融问题、候选回答和金融结论的事实核查。

## 当前目录结构

```text
tracingclaw_finance/
├── tracingclaw_fince_skill.zip        # 目标 Skill：金融验真 Skill
├── westock-data.zip                   # 工具 Skill：结构化金融数据查询
├── mx-finance-search.zip              # 工具 Skill：金融资讯/公告/研报搜索
├── readme.md                          # 本说明
└── 参考示例-仅供模板参考/
    ├── ideal_state.md                 # 理想态参考
    └── testcases.yaml                 # 简化测试用例参考
```

`参考示例-仅供模板参考/` 里的文件只用于说明期望行为和测试方向，不是必须安装的 Skill 代码。

## 文件说明

| 路径 | 说明 |
|------|------|
| `tracingclaw_fince_skill.zip` | 本题的目标 Skill。学生需要围绕这个金融验真 Skill 设计评估用例集，重点检验它在金融事实核查任务中的输出质量。 |
| `westock-data.zip` | 目标 Skill 可调用的工具 Skill，用于查询结构化金融数据，例如股票、行情、K 线、财报、资金、技术指标、股东、分红、评级、市场等。 |
| `mx-finance-search.zip` | 目标 Skill 可调用的工具 Skill，用于搜索金融资讯、公告、研报、财经新闻、政策动态和事件背景。 |
| `readme.md` | 当前说明文件，介绍材料包内容、目标 Skill 能力边界、依赖工具和注意事项。 |
| `参考示例-仅供模板参考/ideal_state.md` | 理想态参考示例，展示如何描述金融验真 Skill 的能力边界、理想流程、质量标准和红线。 |
| `参考示例-仅供模板参考/testcases.yaml` | 测试用例参考示例，展示 YAML 用例的大致结构和几个典型核查场景。这里的用例数量较少，只用于示范，不是最终交付标准。 |

## 可用能力

- 使用 `westock-data` 查询股票、行情、K 线、财报、资金、技术指标、股东、分红、评级、市场等结构化数据。
- 使用 `mx-finance-search` 搜索公告、研报、财经新闻、政策动态和事件背景。
- 对用户提供的金融回答做 0/1/2 分真实性评分。
- 输出可复核的修正参考答案，并标注时间、市场、币种、财年等口径。

## 依赖说明

学生应先解压并阅读 `tracingclaw_fince_skill.zip`，理解目标 Skill 的验真流程、输出格式和红线。该目标 Skill 依赖课程提供的两个外部金融工具 Skill：

- `westock-data`：结构化金融数据源。
- `mx-finance-search`：金融资讯、公告、研报和政策搜索源。

使用前请确保 `westock-data.zip` 和 `mx-finance-search.zip` 已按课程要求安装或可被当前环境调用。它们是辅助工具，不是本题要评估的目标 Skill；本题评估对象是 `tracingclaw_fince_skill.zip`。

## 不支持什么

- 不支持内部分享链接或内部搜索链路解析。
- 不支持任何内部链路接口。
- 不要求学生申请内部权限、Token 或日志。

如果用户只提供内部分享链接，应让用户粘贴原始 question 和 answer，然后基于可用金融数据做验真。

## 示例问法

```text
请核查这段回答是否正确：问题是“腾讯 2024 年营收是多少？”回答是“腾讯 2024 年营收 6600 亿元人民币”。
```

```text
帮我验证“贵州茅台今天股价上涨超过 3%”这个结论是否属实。
```

```text
这个金融回答打几分？请给出核查点、数据来源和修正答案。
```
